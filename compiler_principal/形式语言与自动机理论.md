### 三、识别非正则语言

==鸽巢原理==：把n个物品放到m个盒子中，并且n>m，那么至少有一个盒子李装了一个以上的物品。

> ==定理4.8== 泵引理：设L是**无穷**的**正则语言**，那么一定存在某个正整数m，满足，任意w∈L，|w| ≥ m，可以分解成：
>
> ​									w = xyz
>
> 其中：
>
> ​									|xy| ≤ m 	|y|≥1
>
> 对所有的 i = 0，1，2，3....满足：
>
> ​									w~i~ = xy^i^z
>
> 也属于L。

证明：**构造**一个dfa接受语言L，设这个dfa的状态为q0，q1，q2，...，qn-1。现在在L中取任意符号串w，满足|w| ≥ m = n + 1。（已知L是**无穷**，一定可以这样做）

自动机处理w时，经过的状态集合：

​								q~0~,q~1~,q~2~,.....,q~j~,.....q~z~

因为|w| = n+1，所以一定有个状态重复（**鸽巢原理**），设这个状态为r，且r ≤ n。因此这个序列为：

​								q~0~,q~1~,q~2~,.....,==q==~r~,......==q==~r~,.....q~f~

所以**一定存在**w的子串x,y,z，满足：

> ​								δ^*^（q~0~，x）= q~r~
>
> ​								δ^*^（q~r~，y）= q~r~					
>
> ​								δ^*^（q~r~，z）= q~r~			

因此 |xy| ≤ n + 1 = m，|y| ≥ 1，由此，我们可以立刻得到：

> ​								δ^*^（q~0~，xz）= q~f~
>
> ​								δ^*^（q~r~，xyz）= q~f~					
>
> ​								δ^*^（q~r~，xy^i^z）= q~f~	

使用泵引理证明语言**不是**正则的步骤：

> 1、对手选一个m
>
> 2、对给定的m，在L**任意**挑w，只要满足w∈L，并且**|w| ≥ m**。
>
> 3、对手选择==任意==一种分解方式，分解成xyz，要求|xy| ≤ m，|y| ≥ 1。
>
> 4、我们挑选一个i，使得w~i~ ==∉== L

## 第五章 上下文无关语言

### 一、上下文无关文法

正则文法：左部必须是一个变量，右部有**固定**形式

上下文无关文法：左部必须是一个变量，右部可以是**任何**形式

> ==定义5.1== 文法G = (V，T，S，P)，如果P中的**产生式**具有如下形式：
>
> ​											==A -> x==
>
> 其中，A∈V，x∈(V∪T)^*^ 称为**上下文无关**的

> ==定义5.2== 
>
> **最左**推导：如果每次都替换句型中**最左端**的变量。
>
> **最右**推导：如果每次都替换句型中**最右端**的变量。

推导树：

定义5.3

设G = (V，T，S，P)是一个上下文无关文法。当且仅当下列性质成立时，一个有序树才能称为是G的推导树。

1、**根节点**标记为S。

2、每个**叶节点**的标记 token ∈ T∪{λ}的

3、每个中间节点的标记 token ∈ V

4、如果一个节点有标记**A∈V**，它的**子节点**（从左到右）为a1，a2，a3，...，an，那么，P中一定包含下面形式的**产生式**：

​													A -> a1a2a3...an

5、标记为λ的叶节点没有兄弟，即：一个结点的**子结点**标记为**λ**，那这个结点**只能有**一个子结点。

## 第六章 上下文无关文法的化简与范式

### 一、文法变换方式

1、删除无用的产生式

定义6.1 设 G = （V,T,S,P）是一个上下文无关文法，如果变量A∈V是**有用的**，则当且仅当==至少存在==一个w∈L(G) 使得

> S => xAy => w		其中x,y属于 （V∪T）^*^

总而言之，某个变量是==有用的==当且仅当它至少在一个句型推导中出现，而对于不出现在任何句子推导中的变量，我们称之为==无用==。

例如：

> S -> A
>
> A -> aA | λ
>
> B -> bA
>
> 由于B状态无法到达，因此B变量是无用的

2、删除λ表达式

定义6.2 在上下文无关文法中任意形如：A -> λ的产生式称为λ产生式。一个文法生成的语言可以==不包含==λ。

> 定理6.3 设G是任意的上下文无关文法且λ不在L(G)中，则存在一个等价的==不包含λ==表达式的文法==G'==
>
> 化简步骤：
>
> 首先通过如下步骤找到所有G的可空变量的集合V~N~。
>
> 1、对于所有形如A->λ的产生式，将A加入V~N~
>
> 2、对于所有形如B -> A1A2 ... An 的产生式，如果A1、A2...An都属于V~N~，则将B加入V~N~中。
>
> 其次，考察P中所有形如：A-> x1x2...xm 	m≥1，其中xi ∈V∪T。将该产生式使用λ替换所有可能组合而得到的产生式加入==P'==。如果xi和xj是空的，则p'中存在一个用λ替换xi的产生式，一个λ替换xj的产生式，以及一个λ替换xi和xj的产生式，如果所有xi为空，则A-> λ将不被加入到p'中

例如：

> S -> ABaC
>
> A -> BC
>
> B -> b | λ
>
> C -> D | λ
>
> D -> d
>
> 找出可空变量V~N~ = {A，B，C}
>
> S-> ABaC | BaC | AaC | ABa | Aa | Ba | aC | a
>
> A -> BC | B | C
>
> B -> b
>
> C -> D
>
> D -> d

3、清除单位产生式

定义6.3 上下文无关文法中形如：A -> B 的产生式，其中A，B∈V，我们称为单位产生式。

> ==定理6.4== 设G = (V,T,S,P)是一个任意的上下文无关文法且不包含λ产生式，则存在一个文法G‘ = （V',T',S,P'），它与G等价且不存在单位产生式。

例如：

> S -> Aa | B					画出依赖图得出单位产生式：S -> A | B
>
> B -> A | bb																		  B -> A
>
> A -> a | bc | B																	 A -> B
>
> 找出非单位产生式：
>
> S -> Aa
>
> A -> a | bc
>
> B -> bb
>
> 将单位产生式替换成非单位产生式
>
> S -> a | bc |bb
>
> A -> bb
>
> B -> a | bc
>
> 合并上述产生式
>
> S -> a | bc | bb | Aa
>
> A -> a | bc | bb
>
> B -> a | bc | bb

### 二、两个重要的范式

1、乔姆斯基范式：如果一个上下文无关文法的所有产生式都形如

A -> BC 或者 A -> a，其中A，B，C∈V，a∈T，则该文法属于乔姆斯基范式。

> ==定理6.6==：对于任意的上下文无关文法G = (V,T,S,P)其中 λ ∉ L(G)，都存在一个等价的上下文无关文法G' = (V',T',S,P')属于乔姆斯基范式。
